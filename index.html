FAQ

Pertanyaan yang sering ditanyakan

Tidak menemukan jawaban yang kamu cari? Hubungi tim FinBot via WhatsApp.

Bagaimana cara kerja FinBot? ▾
Setelah berlangganan, kamu akan mendapatkan nomor WhatsApp FinBot. Simpan nomornya di kontak, lalu mulai chat seperti biasa — misalnya "makan siang 45rb" atau "gaji masuk 5 juta". FinBot akan otomatis mencatat dan menyimpan transaksimu ke dashboard. Semua data bisa diakses kapan saja melalui browser.
Apakah data keuangan saya aman? ▾
Ya, data kamu tersimpan aman di server terenkripsi. Hanya kamu yang bisa mengakses datamu sendiri menggunakan akun yang sudah terdaftar. Kami tidak menjual atau membagikan data pengguna kepada pihak ketiga manapun.
Apakah perlu install aplikasi tambahan? ▾
Tidak perlu. FinBot berjalan sepenuhnya melalui WhatsApp yang sudah ada di HP kamu, dan dashboard dapat diakses langsung dari browser HP, tablet, maupun laptop — tanpa install apapun.
Format pesan apa yang diterima FinBot? ▾
FinBot memahami bahasa natural sehari-hari dalam Bahasa Indonesia. Kamu bisa ketik bebas seperti "beli kopi 25rb", "transfer ke teman 200ribu", atau "terima gaji 4.5 juta". FinBot akan otomatis mengenali jenis transaksi, nominal, dan kategorinya. Selain teks, FinBot juga bisa menerima foto struk belanja atau nota transaksi.
Apa bedanya paket Lengkap dengan Dashboard Only? ▾
Paket Lengkap memungkinkan kamu mencatat transaksi langsung via chat WhatsApp secara otomatis. Paket Dashboard Only tidak menyertakan fitur bot WhatsApp, namun tetap memberikan akses penuh ke dashboard untuk input manual, visualisasi chart, ekspor laporan, dan target tabungan.
Bisakah saya mengekspor data transaksi? ▾
Bisa. Semua paket FinBot mendukung ekspor laporan dalam format Excel/CSV maupun PDF. Kamu bisa mengunduh laporan bulanan atau seluruh riwayat transaksi kapan saja dari halaman dashboard.
Bagaimana jika saya ingin berhenti berlangganan? ▾
Langganan FinBot tidak perpanjang otomatis. Setelah masa aktif habis, akses akan berhenti dengan sendirinya. Kamu bebas memilih untuk tidak lanjut tanpa dikenakan biaya apapun. Data transaksi tetap bisa kamu unduh sebelum masa aktif berakhir.
Apakah ada dukungan jika saya mengalami kendala? ▾
Ada. Tim FinBot siap membantu via WhatsApp jika kamu mengalami kendala teknis atau pertanyaan seputar penggunaan. Respons biasanya dalam hitungan jam pada hari kerja.
